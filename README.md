# Cafe Duoi Hien Nha 

Live Website: https://cafeduoihiennha.com

## About

Cafe Duoi Hien Nha is a small coffee shop in Vinh, Vietnam, that just recently opened with a vintage vibe.
This website is a modern, responsive coffee shop website built to showcase signature drinks, gallery images, and customer reviews.  
The website focuses on elegant UI design, smooth animations, and mobile responsiveness.
The website also allows the manager to oversee and make changes to their menu and drink management

## Features

- Responsive design (mobile, tablet, desktop)
- Animated marquee review section
- Signature drinks showcase
- Smooth scroll navigation
- Custom domain deployment on Vercel
- Optimized performance & SEO-ready
- CRUD Operations for backend
- Allowing the admin to delete, add, and make changes to the menu in simple clicks

## Tech Stack

- React (Vite)
- React Router
- CSS (Custom styling)
- Vercel (Deployment & Analytics)
- Porkbun (Domain Management)
- Spring (Backend) For backend documentation, read more [here](./backend/README.md).
- PostgreSQL (Database)
