## About

Cafe Duoi Hien Nha is a modern, responsive coffee shop website built for a local caf√© in Vinh, Vietnam. The platform showcases signature drinks, gallery images, and customer reviews, while also providing an admin dashboard for managing menu items and drink availability. The project emphasizes elegant UI design, smooth animations, mobile-first responsiveness, and a structured backend for dynamic content management.

## Tech Stack

- Front-end: React.js, CSS
- Back-end: Spring Boot read more [here](./backend/README.md)
- Database: PostgreSQL, Cloudinary
- Deployments: Vercel, Render, NeonDB, GitHub
- Testing: Postman

## Features

- Responsive design (mobile, tablet, desktop)
- Animated marquee review section
- Signature drinks showcase
- Smooth scroll navigation
- Custom domain deployment on Vercel
- Optimized performance & SEO-ready
- CRUD Operations for backend
- Allowing the admin to delete, add, and make changes to the menu in simple clicks
- Implemented searching and sorting feature for the menu dashboard
